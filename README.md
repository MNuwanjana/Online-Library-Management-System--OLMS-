# Online Library Management System (OLMS) - Web Application Project

Welcome to the repository for our Web Technology Final Assignment! 

The Online Library Management System (OLMS) is a lightweight, web-based application designed to allow students to view, borrow, and return books, while administrators can manage the library's inventory. 

## 🛠️ Technology Stack
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap (Responsive UI)
* **Backend:** PHP
* **Database:** Flat-file JSON (`users.json`, `books.json`, `reviews.json`)

## 👥 Team & Modules
This project is divided into 5 core modules, handled by our team members:

* **Member 1 (Name):** Homepage & User Dashboard (Stats & Recent Activity)
* **Member 2 (Name):** User Authentication (Login & Signup)
* **Member 3 (Name):** Catalog (View & Search Books)
* **Member 4 (Name):** Admin Panel (Add, Edit, Delete Books)
* **Member 5 (Name):** Member Operations (Borrow, Return, & Reviews)
